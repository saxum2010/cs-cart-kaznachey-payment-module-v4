Данный модуль интеграции с kaznachey разработан для CS-Cart 4.0
 
Перед установкой модуля необходимо зарегистрироваться на сайте kaznachey.ua (раздел "ПОДКЛЮЧЕНИЕ"). 
После подписания договора будут выданы: 
-идентификатор мерчанта; 
-секретный ключ мерчанта; 
Переданные данные будут использованы при настройки модуля

Порядок установки модуля:
1. Откройте панель управления вашего сайта;
2. На странице Валют (Администрирование->Валюты /admin.php?dispatch=currencies.manage) Добавьте валюту грн. с кодом (UAH)
3. Модули->Управление сайтом (/admin.php?dispatch=addons.manage) загрузите архив с модулем оплаты;
4. На странице Администрирование->Способы оплаты (/admin.php?dispatch=payments.manage) добавьте новый способ оплаты
На вкладке "Общее" заполните:
	Название: - www.kaznachey.ua (Webmoney, liqpay.com VISA, MasterCard, и др.)
	Процессор: - Kaznachey
На вкладке "Настройки" заполните:
	merchantGuid: -идентификатор мерчанта; 
	merchnatSecretKey: -секретный ключ мерчанта;
Кнопка "Создать" снизу справа
